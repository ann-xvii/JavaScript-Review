Object-oriented programming

Javascript objects are very simple collections of name value pairs.

There are two ways of creating a simple object in Javascript:

  1. var obj = new Object();
  2. var obj = {};

We can also create an object entirely as follows:

obj = { name: { first: 'Gandalf', last: 'the Grey' }, address: 'Middle Earth' };

In OOP, an object is an instance of a class. A class defines the characteristics of the object.

Example:

This is how we declare a class that represents a book:

function Book(title, pages, isbn){
		this.title = title;
		this.pages = pages;
		this.isbn = isbn;
	}

To instantiate this class, we can use the following code: 

		var book = new Book('title', 'pag', 'isbn');

Then we can access its attributes and update them as follows:

		console.log(book.title);
		book.title = 'Generation P';
		// updates the value of the book title
		console.log(book.title); // outputs the updated value

A class can also contain methods(functions):

		Book.prototype.printTitle = function(){
		   console.log(this.title);
		};
		book.printTitle();

We can also declare functions directly inside the class definition as well:

		function Book(title, pages, isbn) {
			this.title = title;
			this.pages = pages;
			this.isbn = isbn;
			this.printIsbn = function(){
				console.log(this.isbn);
			}
		}

In the prototype example, the printTitle function is going to be shared between all instances, and only one copy is going to be created.

When we use class-based definition, each instance of the class will have its own copy of the functions.

Using the protoype method saves memory and processing cost in regards to assigning the function to the instance.

however, you can only declare public functions and properties using the prototype method.
With a class-based definition, you can declare private functions and properties and the other methods inside the class can also access them.

Whenever possible, use the prototype method, unless, you need to keep some properties and methods private.

Ch.2 Arrays

An array is the simplest memory data structure. For this reason, all programming languages have a built-in array data type.

An array stores a sequence of values that are all of the same data type. Although JS allows us to create arrays with values from different data types, we will follow the best practices and consider that we cannot do that (most languages don't have this capability).

Example use of array

Consider that we need to store the avg temp of each month of the year of the city that we live in:

Fibonacci Numbers

In mathematical terms, the sequence Fn of Fibonacci numbers is defined by the recurrence relation Fn = Fn-1 + Fn-2

with seed values F1 = 1, F2 = 1, or F0 = 0, F1 = 1

Example:

Let's say we want to find out the first 20 numbers of the Fibonacci sequence. The first two numbers of the Fibonacci sequence here, the seed values, will be 1 and 2 (chosen starting point of sequence).

(we skip 0 because there is no 0 in the Fibonacci sequence)

Then all we have to do is create the 3rd to 21st number of the sequence (as we have seeded the first two numbers already).

To do so, we can use a loop and assign the sum of the previous two positions of the array to the current position

	// declare and create array
	var fibonacci = [];
	// assign the first 2 numbers of the Fibonacci sequence to the second and third positions of the array
	fibonacci[1] = 1;
	fibonacci[2] = 1;

	// create the third to the 21st number of the sequence, as we know the first 2 pre-seeded numbers already
	for (var i = 3; i < 21; i++) {
		// assign the previous two positions of the array to the current array position
		fibonacci[i] = fibonacci[i-1] + fibonacci[i-2];
	}

	for (var i = 1; i < fibonacci.length; i++) {
		console.log(fibonacci[i]);
	}

This task is well suited for the array data structure.

We cannot simply add new elements to it as we need them

Splice

We can use the splice method to remove an element from an array by simply specifying the position/index we would like to delete from and how many elements we would like to remove:

	numbers.splice(5, 3)

The above code will remove 3 elements starting from index 5 of our array

Splice for Insertions

	numbers.splice(5,0,2,3,4);

The first argument of the method (here, 5) is the index we want to remove or insert elements from.

The second argument is the number of elements we want to remove (in this case, none, 0)

And the third argument (etc) are the values we would like to insert into the array (here, elements 2, 3, and 4).
