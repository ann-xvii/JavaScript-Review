What about ===?

Type and value check.

When comparing Objects, === will only yield true if x and y reference the same object.


Example:
var person1 = {name: 'John'};
var person2 = {name: 'John'};
console.log(person1 === person2);yields false: they are different objects.

Object-oriented programming

Javascript objects are very simple collections of name value pairs.

There are two ways of creating a simple object in Javascript:

  1. var obj = new Object();
  2. var obj = {};

We can also create an object entirely as follows:

obj = { name: { first: 'Gandalf', last: 'the Grey' }, address: 'Middle Earth' };
